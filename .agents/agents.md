# Crop Intel — Agent Configuration

## Skills
This repo ships 8 reusable skills for autonomous engineering:

1. **repo-audit** — Discover codebase architecture
2. **assumption-challenge** — Challenge product assumptions
3. **functional-validation** — Run all tests/builds/lints
4. **crop-price-validation** — Trace price data end-to-end
5. **clean-code-refactor** — Remove dead code, fix typing
6. **github-readiness** — Prepare for public GitHub
7. **security-privacy-audit** — Audit auth, data, API security
8. **release-readiness** — Final pre-release checklist
